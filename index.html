Welcome

Author: Imogen Boyle

Controllers of yesteryear had a very humble beginning, with basic design such as the joystick, they have evolved greatly to become amazing devices we use as commonplace today. Complex action and functions are easy and quick with the more advanced controllers in use today. These more advanced controllers allow for intricate games, drawing and many other tasks to be created limited by only the imagination thanks to these amazing tools. However, even outstanding creations like these have to start somewhere. Take a look further on this website to learn about different controllers created over many years. Learn about how much they've changed in design and become more user-friendly.
